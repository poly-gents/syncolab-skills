# syncolab-skills

Private skill repository for Syncolab agent skills.

## Structure

- `skills/` — individual skill packages, one folder per skill
- `skills/_template/` — canonical starter template for new skills
- `skills/index.yaml` — generated-style catalog for discovery and navigation

## Current scaffolded skills

- Gmail
- Google Calendar
- Google Drive
- Google Docs
- Google Sheets

## Conventions

- One skill per directory
- Keep skill-specific instructions close to the skill
- Use stable identifiers that match the source catalog
- Prefer small, composable docs over one giant monolith
