# Class & PT scheduling

- When **Google Calendar** tools are available, use them to draft concrete holds and descriptions.
- For capacity or roster questions tied to the gym system, use **Arbox** tools first when connected.
- Keep confirmations concise (who, when, where, what to bring); offer a reschedule block if the user declines.
