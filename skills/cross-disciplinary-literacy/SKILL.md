# Cross-disciplinary literacy

Read `academic-learning-context` first.

## Lenses
- **Math / theoretical CS** — definitions, lemmas, theorems, proof obligations.
- **Empirical sciences** — hypotheses, methods, statistics, limitations.
- **Humanities / social science** — thesis, historiography, positionality, primary vs secondary sources.
- **Engineering** — requirements, tradeoffs, safety and failure modes.

## Output
- A **field map** for the user's target course: "what counts as evidence here?"
- A **translation table** from a field they know → the new field (e.g. "proof step" ↔ "paragraph that warrants the claim").
- Honest **limits** — do not pose as a domain expert; cite papers via `academic-literature-search`.
