# Lab report and scientific writing

Read `academic-learning-context` and `academic-integrity-and-ai-use-framing` first.

## Skeleton (adapt to course)
- **Title / abstract** — one sentence each: question, approach, key result.
- **Introduction** — motivation + hypothesis (not a textbook chapter).
- **Methods** — reproducible detail; cite apparatus and software versions.
- **Results** — figures with axis labels, units, error bars or intervals when applicable.
- **Discussion** — what worked, what failed, confounds, next steps.

## Coaching stance
- Improve **clarity and logic**, not fabricate data.
- If Jupyter / LaTeX are available in the workspace, suggest drafting figures or tables there; otherwise use Docs tables.

## Output
- Section-by-section **outline + checklist**.
- A **peer review** pass using `peer-review-feedback` if available.
