# Learning objectives and Bloom alignment

Read `academic-learning-context` first.

## Writing objectives
- Use measurable verbs: *calculate, justify, design, compare, critique* — not *understand, know, appreciate*.
- One **observable behavior** per objective; avoid laundry lists.

## Bloom quick map
- **Remember / Understand** — recall facts, explain in own words.
- **Apply / Analyze** — use procedure in new context; break argument into parts.
- **Evaluate / Create** — judge sources; synthesize or build artifact.

## Alignment check
- Every **activity** should exercise at least one objective.
- Every **assessment** should map to objectives explicitly (coverage matrix).

## Output
- Objectives table + activity/assessment mapping grid.
- Flag misaligned items (e.g. only "remember" tested while objectives promise "create").
