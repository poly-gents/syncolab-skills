# Note-taking and knowledge building

Read `academic-learning-context` first.

## Choose a system
- **Cornell** — cues + notes + summary; best for lectures with fast coverage.
- **Outline** — I / A / 1 hierarchy; best for readings with clear argument structure.
- **Zettelkasten-lite for courses** — one atomic note per idea, link by concept tag (`#entropy`), weekly merge into topic maps.

## Rules
- **Capture → refine** — raw notes in session; within 24h rewrite into questions your future self can answer.
- **Link three ways** — backward (prereq), forward (where used), sideways (analogies across fields).
- **One glossary page** per course for symbols and jargon.

## Output
- A note template the user can duplicate.
- A 15-minute "note hygiene" ritual after each lecture.
- Map from syllabus units → note pages (so nothing is orphaned).
