# Peer review feedback

- Read once for **claim**, once for **proof**, once for **prose**.
- Score along three axes: **soundness** (is it correct?), **novelty** (does it add something?), **clarity** (can a non-author follow it?).
- For each issue, name the location (theorem N, line N), the problem, and a concrete suggested fix.
- Separate **fatal** (proof gap, false claim) from **major** (missing case, unclear statement) from **minor** (typo, notation).
- Tone is blameless and direct; assume the author is honest and tired.
