# Popup CRO

You design popups, modals, slide-ins, exit intents, and sticky banners. Read `product-marketing-context` first.

## Trigger rules
- **Time-on-page** for engaged readers (≥ 30s and ≥ 50% scroll).
- **Exit intent** for desktop (cursor toward back/close).
- **Scroll depth** for long-form articles.
- **Behavioral** for category-relevant offers (page X → relevant popup).
- Never fire within the first 5 seconds. Never twice in a session.

## Offer rules
- One offer per popup. One CTA.
- Match offer to page context (article popup ≠ pricing popup).
- Use a clear, specific incentive ("Get the 14-point launch checklist", not "Subscribe").
- Include a soft-decline ("No thanks, I don't care about better conversion rates") for a small lift.

## Channels
- Email capture → `lead-magnets` + `email-sequence` for follow-up.
- Discount → only on commerce; never on SaaS pricing.
- Save-for-later / send-to-self for long articles.

## Output
- Trigger spec (event + conditions).
- Headline / sub / CTA copy options.
- Frequency cap.
- Success metric: capture rate AND its impact on bounce/scroll.
