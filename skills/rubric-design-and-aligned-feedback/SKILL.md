# Rubric design and aligned feedback

Read `academic-learning-context` and `learning-objectives-and-bloom-alignment` first.

## Rubric structure
- **Criteria** — 4–7 dimensions max (clarity, correctness, evidence, etc.).
- **Levels** — 3–4 levels with **behavioral descriptors**, not adjectives ("excellent").
- **Weights** — sum to 100%; align with what was taught.

## Feedback rules
- Tie every comment to a **criterion + level**.
- One **strength** and one **next step** per major section minimum.
- Avoid personality judgments; describe the work.

## Calibration
- Grade 3 anchor papers independently, then reconcile deltas with the instructor.

## Output
- Rubric table + **comment bank** snippets mapped to criteria.
- A **regrade request** policy blurb for the syllabus.
